# Diagnostic Frameworks & Decision Trees

## Master Diagnostic: "My Brand Is Struggling"

```
What's the primary symptom?
│
├─ Revenue is flat or declining
│   └─ Go to: Revenue Diagnostic
│
├─ Profitable but not growing
│   └─ Go to: Growth Diagnostic
│
├─ Growing but not profitable
│   └─ Go to: Unit Economics Diagnostic
│
├─ Running out of cash
│   └─ Go to: Cash Flow Emergency
│
└─ Don't know what's wrong
    └─ Go to: Full Health Check
```

## Revenue Diagnostic

```
Revenue flat or declining?
│
├─ Check: Has traffic dropped?
│   │
│   ├─ YES → Traffic Problem
│   │   ├─ Paid traffic down?
│   │   │   ├─ Budget cut? → Reallocate or find efficiency
│   │   │   ├─ CPM increased? → Creative fatigue, test new ads
│   │   │   └─ Platform changes? → Diversify channels
│   │   │
│   │   ├─ Organic traffic down?
│   │   │   ├─ Google update? → Check rankings, audit content
│   │   │   ├─ Social algorithm change? → Adapt content strategy
│   │   │   └─ Seasonal? → Compare YoY, not MoM
│   │   │
│   │   └─ Email traffic down?
│   │       ├─ List shrinking? → Fix acquisition, reduce churn
│   │       ├─ Open rates down? → Clean list, test subjects
│   │       └─ Click rates down? → Content/offer problem
│   │
│   └─ NO → Conversion Problem
│       └─ Go to: Conversion Diagnostic
│
└─ Is it a new customer or repeat customer problem?
    │
    ├─ New customer revenue down
    │   ├─ Check CAC trend → Rising? Creative fatigue, audience saturation
    │   ├─ Check conversion rate → Lower? Site/offer problem
    │   └─ Check traffic quality → Worse? Targeting too broad
    │
    └─ Repeat customer revenue down
        ├─ Check repeat rate trend → Lower? Retention problem
        ├─ Check AOV trend → Lower? Product/pricing issue
        └─ Check churn (if subscription) → Higher? Experience problem
```

## Conversion Diagnostic

```
Conversion rate dropped or below benchmark?
│
├─ Where is the drop-off?
│   │
│   ├─ Homepage to PDP
│   │   ├─ Navigation confusing? → Simplify menu
│   │   ├─ Value prop unclear? → Test hero messaging
│   │   ├─ Page speed slow? → Optimize images, code
│   │   └─ Mobile experience broken? → Test on actual devices
│   │
│   ├─ PDP to Add to Cart
│   │   ├─ Price shock? → Test anchoring, show value
│   │   ├─ Not enough info? → Better images, descriptions
│   │   ├─ No trust signals? → Add reviews, guarantees
│   │   ├─ Out of stock issues? → Fix inventory, show alternatives
│   │   └─ Add to cart button buried? → Make it sticky
│   │
│   ├─ Add to Cart to Checkout
│   │   ├─ Surprise shipping costs? → Show earlier or free threshold
│   │   ├─ No guest checkout? → Add it immediately
│   │   ├─ Too many steps? → Streamline to one page
│   │   └─ Cart abandonment? → Implement recovery flows
│   │
│   └─ Checkout to Purchase
│       ├─ Payment issues? → Add more options (Apple Pay, PayPal)
│       ├─ Security concerns? → Add trust badges
│       ├─ Form errors? → Test full checkout flow
│       └─ Last-minute hesitation? → Add guarantee, support
│
└─ Device-specific issues?
    ├─ Mobile-only drop? → Mobile UX audit required
    └─ Desktop-only drop? → Less common, check recent changes
```

## Unit Economics Diagnostic

```
Not profitable or margins declining?
│
├─ Check: Contribution Margin
│   │
│   │ Formula: (Revenue - COGS - Shipping - Payment Fees) / Revenue
│   │ Benchmark: Should be 50-70%
│   │
│   ├─ Below 50%?
│   │   ├─ COGS too high?
│   │   │   ├─ Renegotiate supplier terms
│   │   │   ├─ Find alternative suppliers
│   │   │   ├─ Increase order volumes for discount
│   │   │   └─ Raise prices (test first)
│   │   │
│   │   ├─ Shipping eating margin?
│   │   │   ├─ Negotiate better rates
│   │   │   ├─ Optimize packaging (dimensional weight)
│   │   │   ├─ Consolidate shipments
│   │   │   └─ Raise free shipping threshold
│   │   │
│   │   └─ Payment fees high?
│   │       ├─ Shop Shop Pay vs Stripe vs PayPal
│   │       └─ Negotiate volume discounts
│   │
│   └─ Contribution margin healthy but still not profitable?
│       └─ Check: CAC and Marketing Efficiency
│
├─ Check: CAC Ratio
│   │
│   │ Formula: LTV / CAC
│   │ Benchmark: Should be 3:1 or better
│   │
│   ├─ Below 3:1?
│   │   ├─ CAC too high
│   │   │   ├─ Creative fatigue → New ads needed
│   │   │   ├─ Audience exhaustion → New channels
│   │   │   ├─ Inefficient spend → Cut underperformers
│   │   │   └─ Competition increased → Find differentiation
│   │   │
│   │   └─ LTV too low
│   │       ├─ Low AOV → Bundle, upsell, cross-sell
│   │       ├─ Low repeat rate → Retention flows
│   │       └─ High churn → Product/experience issue
│   │
│   └─ Ratio healthy but not profitable?
│       └─ Check: Fixed Costs
│
└─ Check: Fixed Cost Ratio
    │
    │ Formula: Fixed Costs / Revenue
    │ Benchmark: Should be <30%
    │
    └─ Above 30%?
        ├─ Payroll too high? → Evaluate team efficiency
        ├─ Office/warehouse costs? → Consider alternatives
        ├─ Software bloat? → Audit subscriptions
        └─ Debt service? → Refinance or pay down
```

## Cash Flow Emergency

```
Running out of cash? (Runway < 60 days)
│
├─ IMMEDIATE ACTIONS (Do today):
│   ├─ Pause all non-essential ad spend
│   ├─ Calculate exact runway (Cash ÷ Monthly burn)
│   ├─ List all payables and due dates
│   └─ Identify what can be delayed
│
├─ WEEK 1 PRIORITIES:
│   │
│   ├─ Revenue acceleration
│   │   ├─ Flash sale to existing customers (40-50% off)
│   │   ├─ Winback campaign to lapsed customers
│   │   ├─ Push subscription upgrades
│   │   └─ Pre-sell upcoming products
│   │
│   ├─ Payables management
│   │   ├─ Call suppliers, request Net 60 or 90
│   │   ├─ Negotiate payment plans
│   │   ├─ Prioritize critical vendors
│   │   └─ Delay non-critical invoices
│   │
│   └─ Expense cuts
│       ├─ Cancel unused software
│       ├─ Pause contractor work
│       ├─ Negotiate rent deferral
│       └─ Cut nice-to-haves
│
├─ WEEK 2-4: FINANCING OPTIONS
│   │
│   ├─ Revenue-based financing
│   │   ├─ Clearco (12-24 hr funding)
│   │   ├─ Wayflyer (ecom focused)
│   │   ├─ Shopify Capital (if on Shopify)
│   │   └─ Payability (Amazon sellers)
│   │
│   ├─ Inventory financing
│   │   ├─ Use inventory as collateral
│   │   └─ Sell slow inventory at cost
│   │
│   └─ Bridge options
│       ├─ Founder contribution
│       ├─ Friends/family bridge
│       ├─ Existing investor bridge
│       └─ Revenue share deal
│
└─ IF ALL ELSE FAILS:
    ├─ Wind down gracefully (pay employees, vendors)
    ├─ Sell assets (inventory, domain, customer list)
    ├─ Strategic acquisition (sell the business)
    └─ Formal process (if debts exceed assets)
```

## Growth Diagnostic

```
Profitable but not growing?
│
├─ Is the market saturated?
│   │
│   ├─ YES → Expansion needed
│   │   ├─ New products for existing customers
│   │   ├─ New customer segments
│   │   ├─ New geographies
│   │   ├─ New channels (retail, Amazon, international)
│   │   └─ Adjacent categories
│   │
│   └─ NO → Execution problem
│       └─ Check: Why aren't you scaling?
│
├─ Why aren't you scaling?
│   │
│   ├─ Can't scale ads profitably?
│   │   ├─ Creative ceiling → More creative volume
│   │   ├─ Audience ceiling → New platforms
│   │   ├─ CAC ceiling → Improve unit economics
│   │   └─ Budget ceiling → Test incrementality
│   │
│   ├─ Operational constraints?
│   │   ├─ Inventory limits → Improve forecasting, financing
│   │   ├─ Fulfillment limits → Better 3PL, more capacity
│   │   └─ Team limits → Hire or outsource
│   │
│   └─ Strategic constraints?
│       ├─ Founder burnout → Build team, delegate
│       ├─ Capital constraints → Consider raising
│       └─ Vision unclear → Define 3-year roadmap
│
└─ What's the unlock?
    ├─ More products → Product expansion roadmap
    ├─ More channels → Channel diversification plan
    ├─ More team → Hiring plan with ROI
    └─ More capital → Fundraising or debt strategy
```

## AOV Diagnostic

```
AOV too low or declining?
│
├─ Current tactics in place?
│   │
│   ├─ Free shipping threshold?
│   │   ├─ NO → Add immediately (set at 30% above current AOV)
│   │   └─ YES but not working → Adjust threshold or messaging
│   │
│   ├─ Upsells/cross-sells?
│   │   ├─ PDP cross-sells? → "Complete the look" or "Pairs well with"
│   │   ├─ Cart upsells? → "Add X for $Y" one-click
│   │   ├─ Post-purchase upsells? → "Wait! Add this before we ship"
│   │   └─ Tools: Rebuy, One Click Upsell, CartHook
│   │
│   ├─ Bundles?
│   │   ├─ Starter bundles → Entry point + essentials
│   │   ├─ Value bundles → More product, better per-unit
│   │   └─ Curated bundles → "Expert picks" themed
│   │
│   └─ Quantity breaks?
│       ├─ Buy 2 get 10% off
│       ├─ Buy 3 get 20% off
│       └─ Subscribe & save
│
├─ AOV by traffic source?
│   │
│   ├─ Email > Paid > Organic is healthy
│   ├─ If paid AOV low → Offer in ads may attract bargain hunters
│   └─ If organic AOV low → SEO bringing wrong audience
│
└─ AOV by device?
    ├─ Desktop > Mobile is normal
    └─ If mobile way lower → Mobile checkout friction
```

## Repeat Purchase Diagnostic

```
Repeat rate below 25%?
│
├─ Do you have post-purchase flows?
│   │
│   ├─ Order confirmation → Brand story, set expectations
│   ├─ Shipping notification → Track link, excitement
│   ├─ Delivered → How to use, tips
│   ├─ Day 5-7 check-in → "How's it going?" + support
│   ├─ Day 10-14 review ask → Build social proof
│   └─ Day 25-30 replenishment → "Time for more?"
│   │
│   └─ If NO to any → Build them immediately
│
├─ Is your product replenishable?
│   │
│   ├─ YES → Subscription strategy
│   │   ├─ Subscription option on PDP?
│   │   ├─ Discount for subscribing?
│   │   ├─ Easy skip/pause?
│   │   └─ Pre-delivery reminders?
│   │
│   └─ NO → Cross-sell strategy
│       ├─ Complementary products?
│       ├─ New product launches?
│       ├─ Limited editions?
│       └─ Loyalty program?
│
├─ Have you surveyed churned customers?
│   │
│   └─ Send: "We miss you - what happened?"
│       ├─ Price issue → Offer incentive
│       ├─ Product issue → Fix and win back
│       ├─ Forgot about you → Better flows
│       └─ Switched competitor → Learn why
│
└─ Winback campaigns active?
    ├─ 30 days since purchase → "Miss you" + small offer
    ├─ 60 days → "It's been a while" + better offer
    ├─ 90 days → "Last chance" + best offer
    └─ 120+ days → Sunset or re-engage sequence
```

## Full Health Check Framework

Run through each section, score 1-10:

### 1. Acquisition (CAC, ROAS, Traffic)
- [ ] CAC stable or declining?
- [ ] ROAS at or above target?
- [ ] Traffic diversified (not >50% one source)?
- [ ] Creative pipeline healthy (3+ new/week)?
- [ ] Testing budget allocated (10% of spend)?

### 2. Conversion (CVR, Add to Cart, Checkout)
- [ ] Site conversion at benchmark?
- [ ] Mobile experience optimized?
- [ ] Page speed under 3 seconds?
- [ ] Checkout streamlined?
- [ ] Trust signals in place?

### 3. Retention (Repeat Rate, LTV, Churn)
- [ ] Repeat rate at benchmark?
- [ ] Post-purchase flows built?
- [ ] Subscription/replenishment in place?
- [ ] Winback campaigns active?
- [ ] NPS/satisfaction tracked?

### 4. Unit Economics (Margin, AOV, Payback)
- [ ] Contribution margin >50%?
- [ ] LTV/CAC >3?
- [ ] CAC payback <90 days?
- [ ] AOV growing or stable?
- [ ] Margin improving?

### 5. Operations (Inventory, Fulfillment, Support)
- [ ] Inventory forecasting accurate?
- [ ] Fulfillment under 48 hours?
- [ ] Support response under 24 hours?
- [ ] Returns rate under benchmark?
- [ ] CSAT/support quality tracked?

### 6. Cash & Finance (Runway, AR/AP, Debt)
- [ ] Runway >6 months?
- [ ] Cash conversion cycle healthy?
- [ ] Debt manageable?
- [ ] Revenue growing >15% YoY?
- [ ] Path to profitability clear?

**Scoring:**
- 50-60: Excellent health
- 40-49: Good with room for improvement
- 30-39: Significant issues to address
- <30: Critical - focus on survival
