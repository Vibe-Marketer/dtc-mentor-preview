# DTC Mentor AI

AI-powered ecommerce mentor chatbot for direct-to-consumer brand owners.

**Domain:** dtcmentor.ai

## Quick Start

Open `index.html` in a browser. No build step, no dependencies, no backend.

## Architecture

- **index.html** — Single page app structure
- **styles.css** — Premium dark mode design system (Inter font, purple accent)
- **script.js** — Chat engine with keyword-based topic routing and knowledge base

## Features

- Email capture gate (initial + after 3 messages for anonymous users)
- Smart topic detection routing across 10 DTC categories
- Pre-built expert responses with real benchmarks and tactical advice
- Typing animation, smooth transitions, session persistence
- Mobile-first responsive design
- Intersection Observer scroll animations

## Knowledge Base Topics

CAC, Facebook/Meta Ads, Email Marketing, Retention/LTV, ROAS, Scaling, Black Friday/BFCM, Product Research, Paid Media Strategy, AOV Optimization

## Deployment

Static files — deploy to any CDN, Vercel, Netlify, or Cloudflare Pages.
